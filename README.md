# 🚀 Care Transition Intelligence System

An advanced multi-page analytics dashboard built using **Streamlit** to monitor, analyze, and forecast care transition efficiency from **CBP custody to HHS placement**.

---

## 📌 Overview

This project provides a **data-driven decision support system** that helps in:

- Monitoring real-time care transition metrics  
- Identifying bottlenecks and delays  
- Analyzing trends and distributions  
- Managing operational data  
- Forecasting future patterns using ML  

---

## 🧠 Key Features

### 📊 Dashboard
- Real-time KPI monitoring  
- Transfer Efficiency & Discharge Effectiveness  
- Backlog and Load analysis  
- Interactive visualizations  

### 📈 Analysis
- Advanced data exploration  
- Trend & distribution analysis  
- Correlation heatmaps  
- Insights generation  

### 🤖 Predictions
- Machine Learning-based forecasting  
- Future trend prediction  
- Data-driven decision support  

### 🗄️ Data Manager
- Add / Edit / Delete records  
- Data cleaning operations  
- Bulk deletion support  
- Interactive data table  

---
